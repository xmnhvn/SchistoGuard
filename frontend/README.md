
  # schistoguard (Community)

  This is a code bundle for schistoguard (Community). The original project is available at https://www.figma.com/design/Q1BIWu12rZGKP1gnjpjKeA/schistoguard--Community-.

  ## Running the code

  Run `npm i` to install the dependencies.

  ## Dev modes (Option 1: Local Frontend + Cloud Backend)

  - `npm run dev` or `npm run dev:cloud`: Frontend runs locally and uses Railway backend.
  - `npm run dev:local`: Frontend runs locally and uses local backend at `http://localhost:3001`.

  ## Build modes

  - `npm run build:cloud`: Build frontend targeting cloud backend.
  - `npm run build:local`: Build frontend targeting local backend.
